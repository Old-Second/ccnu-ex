# 华师制霸生成器

华中师范大学制霸标记生成工具

http://ccnu-ex.muxixyz.com/
https://www.oldsecond.cn/ccnu-ex/

## 功能

可以在线标记 每日必去、经常出没、偶尔会去、只是路过、从未踏足 标记后可生成图片进行分享

标记过的信息会记录到本地存储，有什么新进展再次打开网页更新即可

## 提示

请使用自带浏览器进行保存
